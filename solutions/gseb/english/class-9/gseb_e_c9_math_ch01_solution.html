1.1 Irrational NumbersIn-text concept checks and worked examples

📘 Key Definition

A rational number can be written as \(\frac{p}{q}\), where \(p,q\in\mathbb Z\) and \(q\ne0\).

✏️ Step-by-Step Solution
  1. Let the integer be \(n\).
  2. Write it as \[n=\frac n1\]
  3. Both \(n\) and \(1\) are integers, and the denominator is non-zero.
✅ Final Answer

Yes. Every integer \(n\) is rational because \(n=\frac n1\).

🌐 Gujarati Translation

હા. દરેક પૂર્ણાંક \(n\) પરિમેય છે, કારણ કે \(n=\frac n1\).

📘 Formula Used

The average of two different rational numbers lies between them.

\[\text{Middle number}=\frac{a+b}{2}\]

✏️ Step-by-Step Solution
  1. Take the average: \[\frac{\frac13+\frac12}{2}\]
  2. Add the fractions: \[\frac13+\frac12=\frac56\]
  3. Divide by \(2\): \[\frac{5/6}{2}=\frac5{12}\]
  4. Check: \[\frac13=\frac4{12}<\frac5{12}<\frac6{12}=\frac12\]
✅ Final Answer

One rational number is \(\boxed{\frac5{12}}\).

🌐 Gujarati Translation

એક પરિમેય સંખ્યા \(\boxed{\frac5{12}}\) છે.

🎯 To Prove

\(\sqrt2\) cannot be expressed as \(\frac{p}{q}\) in lowest terms.

✏️ Step-by-Step Proof
  1. Assume \(\sqrt2=\frac pq\), where \(p,q\) are coprime integers and \(q\ne0\).
  2. Square both sides: \[2=\frac{p^2}{q^2}\]
  3. Therefore, \[p^2=2q^2\] so \(p^2\) is even and hence \(p\) is even.
  4. Let \(p=2k\). Then \[(2k)^2=2q^2\]
  5. So \[4k^2=2q^2\Rightarrow q^2=2k^2\]
  6. Thus \(q\) is also even.
  7. Both \(p\) and \(q\) are even, contradicting that they are coprime.
✅ Final Answer

Therefore, \(\boxed{\sqrt2\text{ is irrational}}\).

🌐 Gujarati Translation

અેથી \(\boxed{\sqrt2\text{ અપરિમેય છે}}\).

📘 Key Rule

A rational number has a terminating or eventually recurring decimal expansion.

✏️ Step-by-Step Solution
  1. The number of zeros between successive 1s keeps increasing.
  2. Therefore, no fixed block of digits repeats forever.
  3. The decimal neither terminates nor recurs.
✅ Final Answer

The number is \(\boxed{\text{irrational}}\).

🌐 Gujarati Translation

આ સંખ્યા \(\boxed{\text{અપરિમેય}}\) છે.

1.2 Real Numbers and Their Decimal ExpansionsRecurring decimals and denominator test

📋 Given

Let \(x=0.272727\ldots\).

✏️ Step-by-Step Solution
  1. Because two digits repeat, multiply by \(100\): \[100x=27.272727\ldots\]
  2. Subtract \(x=0.272727\ldots\): \[100x-x=27\]
  3. Thus \[99x=27\]
  4. Divide by \(99\): \[x=\frac{27}{99}=\frac3{11}\]
✅ Final Answer

\(\boxed{0.\overline{27}=\frac3{11}}\).

🌐 Gujarati Translation

\(\boxed{0.\overline{27}=\frac3{11}}\).

📘 Denominator Test

In lowest form, \(\frac{p}{q}\) terminates exactly when \(q=2^m5^n\) for non-negative integers \(m,n\).

✏️ Step-by-Step Solution
  1. The fraction \(\frac{13}{160}\) is already in lowest terms.
  2. Factor the denominator: \[160=2^5\times5\]
  3. Only the prime factors \(2\) and \(5\) occur.
  4. Hence the decimal expansion terminates.
  5. Indeed, \[\frac{13}{160}=0.08125\]
✅ Final Answer

It has a \(\boxed{\text{terminating decimal expansion}}\).

🌐 Gujarati Translation

તેનું દશાંશ સ્વરૂપ \(\boxed{\text{સાંત}}\) છે.

1.3 Representing Real Numbers on the Number LineGeometrical constructions

✏️ Step-by-Step Construction
  1. Mark \(O=0\) and \(A=1\) on the number line.
  2. At \(A\), draw \(AB\perp OA\) with \(AB=1\).
  3. Join \(OB\). Then \[OB=\sqrt{1^2+1^2}=\sqrt2\]
  4. At \(B\), draw a unit segment perpendicular to \(OB\), ending at \(C\).
  5. Then \[OC=\sqrt{OB^2+1^2}=\sqrt{2+1}=\sqrt3\]
  6. With centre \(O\) and radius \(OC\), cut the number line.
✅ Final Answer

The point cut by the arc represents \(\boxed{\sqrt3}\).

🌐 Gujarati Translation

ચાપથી મળતું બિંદુ \(\boxed{\sqrt3}\) દર્શાવે છે.

✏️ Step-by-Step Activity
  1. Draw \(OP_1=1\) unit.
  2. At \(P_1\), draw \(P_1P_2=1\) unit perpendicular to \(OP_1\). Then \(OP_2=\sqrt2\).
  3. At \(P_2\), draw \(P_2P_3=1\) unit perpendicular to \(OP_2\). Then \(OP_3=\sqrt3\).
  4. Continue the same construction. Each new hypotenuse has length \(\sqrt n\).
📊 Square Root Spiral
✅ Final Answer

The construction forms the square root spiral.

🌐 Gujarati Translation

આ રચનાથી વર્ગમૂળ સર્પિલ બને છે.

1.4 Operations on Real NumbersSurds, rationalisation and exponents

📘 Formula Used

Multiply by the conjugate and use \((a+b)(a-b)=a^2-b^2\).

✏️ Step-by-Step Solution
  1. The conjugate of \(\sqrt3+\sqrt2\) is \(\sqrt3-\sqrt2\).
  2. Multiply numerator and denominator: \[\frac1{\sqrt3+\sqrt2}\times\frac{\sqrt3-\sqrt2}{\sqrt3-\sqrt2}\]
  3. Simplify the denominator: \[(\sqrt3)^2-(\sqrt2)^2=3-2=1\]
  4. Therefore, \[\frac1{\sqrt3+\sqrt2}=\sqrt3-\sqrt2\]
✅ Final Answer

\(\boxed{\sqrt3-\sqrt2}\).

🌐 Gujarati Translation

\(\boxed{\sqrt3-\sqrt2}\).

📘 Law Used

\(a^{m/n}=(\sqrt[n]{a})^m\).

✏️ Step-by-Step Solution
  1. Write \(27=3^3\).
  2. Then \[27^{2/3}=(3^3)^{2/3}\]
  3. Multiply the exponents: \[=3^{3\cdot2/3}\]
  4. Therefore \[=3^2=9\]
✅ Final Answer

\(\boxed{9}\).

🌐 Gujarati Translation

\(\boxed{9}\).

📘 Definition

A number is rational if it can be written as \(\frac{p}{q}\) with integers \(p,q\) and \(q\ne0\).

✏️ Step-by-Step Solution

  1. The measured values of circumference and diameter are not exact real-number values; they are finite approximations.
  2. A quotient of two measured decimal approximations is rational, but it only approximates \(\pi\).
  3. The exact geometric lengths \(c\) and \(d\) need not both be rational numbers.
  4. Therefore \(\frac cd=\pi\) can be irrational without contradiction.
✅ Final Answer

There is no contradiction: measurements are approximate, while the exact ratio \(\frac cd\) is the irrational number \(\pi\).
