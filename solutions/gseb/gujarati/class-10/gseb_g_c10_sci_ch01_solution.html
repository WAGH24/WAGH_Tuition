📘 ઉપયોગ: આ પૃષ્ઠમાં માત્ર પાઠ્યપુસ્તકના 8 અંદરના પ્રશ્નો અને સ્વાધ્યાયના 20 પ્રશ્નોના વિગતવાર ઉકેલો મૂળ ક્રમમાં આપવામાં આવ્યા છે. એક સમયે માત્ર એક જ ઉકેલ ખુલશે.

PART I — પાઠની અંદરના પ્રશ્નો

પાઠ્યપુસ્તકના અંદરના પ્રશ્નો — પ્રશ્ન 1 થી 8

✅ જવાબ

મેગ્નેશિયમ પટ્ટીની સપાટી પર હવામાં રહેતાં મેગ્નેશિયમ ઑક્સાઇડનું પાતળું પડ બની જાય છે. આ પડ પટ્ટીને સરળતાથી સળગવા દેતું નથી. સેન્ડપેપરથી ઘસતાં પડ દૂર થાય છે અને શુદ્ધ મેગ્નેશિયમ ઑક્સિજન સાથે સીધી પ્રક્રિયા કરી શકે છે.

💡 મુખ્ય મુદ્દો

સફાઈથી પ્રક્રિયકની સપાટી ખુલ્લી થાય છે અને દહન સમાન રીતે થાય છે.

📘 સમતોલનનો નિયમ

દરેક તત્ત્વના પરમાણુઓની સંખ્યા પ્રક્રિયક અને નીપજ બન્ને બાજુ સમાન રાખવી.

✏️ પગલુંવાર ઉકેલ

(i) હાઇડ્રોજન અને ક્લોરિન

\[\ce{H2 + Cl2 -> 2HCl}\]

(ii) બેરિયમ ક્લોરાઇડ અને એલ્યુમિનિયમ સલ્ફેટ

SO₄ જૂથને એકમ તરીકે ગણીએ: જમણી બાજુ 3BaSO₄ માટે 3BaCl₂ જોઈએ; ત્યારબાદ Cl સંતુલિત કરવા 2AlCl₃ મૂકીએ.

\[\ce{3BaCl2 + Al2(SO4)3 -> 3BaSO4 + 2AlCl3}\]

(iii) સોડિયમ અને પાણી

\[\ce{2Na + 2H2O -> 2NaOH + H2}\]

✅ અંતિમ જવાબ

ઉપરના ત્રણેય સમીકરણોમાં દરેક તત્ત્વના પરમાણુઓની સંખ્યા બન્ને બાજુ સમાન છે.

📘 સંજ્ઞાઓ

(aq) = જલીય દ્રાવણ, (s) = ઘન, (l) = પ્રવાહી.

✏️ પગલુંવાર ઉકેલ

(i) અવક્ષેપન પ્રક્રિયા

\[\ce{BaCl2(aq) + Na2SO4(aq) -> BaSO4(s) v + 2NaCl(aq)}\]

(ii) તટસ્થીકરણ પ્રક્રિયા

\[\ce{NaOH(aq) + HCl(aq) -> NaCl(aq) + H2O(l)}\]

✅ અંતિમ જવાબ

પહેલી પ્રક્રિયામાં સફેદ BaSO₄ અવક્ષેપ મળે છે; બીજી પ્રક્રિયામાં ક્ષાર અને પાણી બને છે.

🎯 ઓળખ

પદાર્થ X = કેલ્શિયમ ઑક્સાઇડ (કળીચૂનો/ક્વિક લાઇમ); સૂત્ર = CaO.

✏️ પ્રક્રિયા

\[\ce{CaO(s) + H2O(l) -> Ca(OH)2(aq)}\; + \;\text{ઉષ્મા}\]

✅ જવાબ

બનતું Ca(OH)₂ ફોરેલો ચૂનો છે. દિવાલ પર લગાવ્યા પછી તે હવામાંના CO₂ સાથે પ્રક્રિયા કરી સફેદ CaCO₃નું પડ બનાવે છે.

\[\ce{Ca(OH)2(aq) + CO2(g) -> CaCO3(s) + H2O(l)}\]

📘 સૂત્ર

પાણીનું અણુ H₂O છે; તેમાં હાઇડ્રોજન અને ઑક્સિજનના પરમાણુઓનું પ્રમાણ 2 : 1 છે.

✏️ સમજણ

\[\ce{2H2O -> 2H2 + O2}\]

સમીકરણ મુજબ 2 આયતન H₂ સામે 1 આયતન O₂ બને છે.

✅ અંતિમ જવાબ

બમણો જથ્થો ધરાવતો વાયુ હાઇડ્રોજન છે.

📘 કારણ

આયર્ન કોપર કરતાં વધુ ક્રિયાશીલ છે. તે CuSO₄માંથી કોપરને વિસ્થાપિત કરી લીલાશ પડતું FeSO₄ બનાવે છે.

✏️ સમીકરણ

\[\ce{Fe + CuSO4 -> FeSO4 + Cu}\]

✅ અંતિમ જવાબ

વાદળી CuSO₄ ઘટે છે અને FeSO₄ બને છે; તેથી દ્રાવણનો રંગ વાદળીમાંથી લીલાશ પડતો થાય છે.

✅ ઉદાહરણ

સિલ્વર નાઇટ્રેટ અને સોડિયમ ક્લોરાઇડ વચ્ચેની પ્રક્રિયા:

\[\ce{AgNO3(aq) + NaCl(aq) -> AgCl(s) v + NaNO3(aq)}\]

🧠 સમજણ

Ag⁺ અને Na⁺ પોતાના ઋણાયનોની આપલે કરે છે અને સફેદ AgCl અવક્ષેપ બને છે.

✏️ (i) વિશ્લેષણ

\[\ce{4Na + O2 -> 2Na2O}\]

Na ઑક્સિજન મેળવે છે તેથી Naનું ઑક્સિડેશન થાય છે. O₂ સોડિયમ સાથે જોડાઈ ઑક્સાઇડ બને છે; O₂નું રિડક્શન થાય છે.

✏️ (ii) વિશ્લેષણ

\[\ce{CuO + H2 -> Cu + H2O}\]

H₂ ઑક્સિજન મેળવી H₂O બને છે, એટલે H₂નું ઑક્સિડેશન. CuO ઑક્સિજન ગુમાવી Cu બને છે, એટલે CuOનું રિડક્શન.

✅ પરિણામ

પ્રક્રિયાઑક્સિડેશન પામે છેરિડક્શન પામે છે
(i)NaO₂
(ii)H₂CuO

PART II — પ્રકરણના અંતે આપેલ સ્વાધ્યાય

સ્વાધ્યાય — પ્રશ્ન 1 થી 20 (મૂળ ક્રમબદ્ધતા)

✅ સાચો જવાબ

(i) (a) અને (b)

🧠 શા માટે?

\(\ce{PbO}\) ઑક્સિજન ગુમાવી \(\ce{Pb}\) બને છે, તેથી લેડ ઑક્સાઇડનું રિડક્શન થાય છે; “લેડ રિડક્શન પામે છે” એવું કહેવું યોગ્ય નથી. \(\ce{C}\) ઑક્સિજન મેળવી \(\ce{CO2}\) બને છે, તેથી Cનું ઑક્સિડેશન થાય છે; CO₂નું ઑક્સિડેશન થતું નથી.

❌ અન્ય વિધાનો

(c) સાચું છે: કાર્બન ઑક્સિડેશન પામે છે.
(d) સાચું છે: PbO રિડક્શન પામે છે.

✅ સાચો જવાબ

(d) વિસ્થાપન પ્રક્રિયા

🧠 શા માટે?

એલ્યુમિનિયમ આયર્ન કરતાં વધુ ક્રિયાશીલ હોવાથી Fe₂O₃માંથી આયર્નને વિસ્થાપિત કરે છે.

\[\ce{Fe2O3 + 2Al -> Al2O3 + 2Fe}\]

❌ અન્ય વિકલ્પો

એક જ નીપજ નથી તેથી સંયોગીકરણ નહીં; એક પદાર્થ તૂટતો નથી તેથી વિઘટન નહીં; આયનોની પરસ્પર આપલે નહીં પરંતુ એક તત્ત્વ બીજા તત્ત્વને દૂર કરે છે, તેથી દ્વિવિસ્થાપન નહીં.

✅ સાચો જવાબ

(a) હાઇડ્રોજન વાયુ અને આયર્ન ક્લોરાઇડ બને છે.

✏️ સમીકરણ

\[\ce{Fe(s) + 2HCl(aq) -> FeCl2(aq) + H2(g) ^}\]

🧠 સમજણ

લોખંડ હાઇડ્રોજન કરતાં વધુ ક્રિયાશીલ હોવાથી તે HClમાંથી હાઇડ્રોજનને વિસ્થાપિત કરે છે.

❌ અન્ય વિકલ્પો

ક્લોરિન વાયુ કે આયર્ન હાઇડ્રૉક્સાઇડ બનતા નથી; પ્રક્રિયા ચોક્કસ થાય છે; પાણી નીપજ નથી.

✅ વ્યાખ્યા

જે રાસાયણિક સમીકરણમાં દરેક તત્ત્વના પરમાણુઓની સંખ્યા પ્રક્રિયક બાજુ અને નીપજ બાજુ સમાન હોય તેને સમતોલિત રાસાયણિક સમીકરણ કહે છે.

🧠 શા માટે સમતોલિત કરવું?

દળ સંરક્ષણના નિયમ અનુસાર રાસાયણિક પ્રક્રિયામાં દળનું સર્જન કે નાશ થતો નથી. તેથી પ્રક્રિયા પહેલાં અને પછી દરેક તત્ત્વના પરમાણુઓની સંખ્યા સમાન હોવી જોઈએ.

💡 ઉદાહરણ

\[\ce{2H2 + O2 -> 2H2O}\]

બન્ને બાજુ H = 4 અને O = 2 છે.

✏️ પગલુંવાર સમીકરણો

(a)

કંકાલ: \(\ce{N2 + H2 -> NH3}\). N સંતુલિત કરવા 2NH₃ અને પછી H માટે 3H₂.

\[\ce{N2(g) + 3H2(g) -> 2NH3(g)}\]

(b)

S માટે 2H₂S અને 2SO₂; H માટે 2H₂O; O માટે 3O₂.

\[\ce{2H2S(g) + 3O2(g) -> 2SO2(g) + 2H2O(l)}\]

(c)

SO₄ જૂથ સંતુલિત કરવા 3BaCl₂; Al માટે 2AlCl₃.

\[\ce{3BaCl2(aq) + Al2(SO4)3(aq) -> 2AlCl3(aq) + 3BaSO4(s) v}\]

(d)

K અને H₂O માટે ગુણાંક 2 મૂકો.

\[\ce{2K(s) + 2H2O(l) -> 2KOH(aq) + H2(g)}\]

✅ અંતિમ જવાબ

બધા સમીકરણો સૌથી નાના પૂર્ણાંક ગુણાંકથી સમતોલિત છે.

✏️ પગલુંવાર ઉકેલ

(a)

Ca(NO₃)₂માં બે નાઇટ્રેટ જૂથ હોવાથી HNO₃ આગળ 2; ત્યારબાદ H₂O આગળ 2.

\[\ce{2HNO3 + Ca(OH)2 -> Ca(NO3)2 + 2H2O}\]

(b)

Na₂SO₄ માટે 2NaOH; કુલ H સંતુલિત કરવા 2H₂O.

\[\ce{2NaOH + H2SO4 -> Na2SO4 + 2H2O}\]

(c)

બધા પરમાણુ પહેલેથી સમાન છે.

\[\ce{NaCl + AgNO3 -> AgCl + NaNO3}\]

(d)

Cl બે હોવાથી HCl આગળ 2.

\[\ce{BaCl2 + H2SO4 -> BaSO4 + 2HCl}\]

✅ ચકાસણી

દરેક સમીકરણમાં બન્ને બાજુ દરેક તત્ત્વના પરમાણુઓ સમાન છે.

✏️ સમતોલિત સમીકરણો

\[\ce{Ca(OH)2(aq) + CO2(g) -> CaCO3(s) + H2O(l)}\]
\[\ce{Zn(s) + 2AgNO3(aq) -> Zn(NO3)2(aq) + 2Ag(s)}\]
\[\ce{2Al(s) + 3CuCl2(aq) -> 2AlCl3(aq) + 3Cu(s)}\]
\[\ce{BaCl2(aq) + K2SO4(aq) -> BaSO4(s) v + 2KCl(aq)}\]

💡 નોંધ

(b) અને (c) વિસ્થાપન છે; (a) અને (d)માં અવક્ષેપ બને છે.

✏️ ઉકેલ

(a) દ્વિવિસ્થાપન

\[\ce{2KBr(aq) + BaI2(aq) -> 2KI(aq) + BaBr2(aq)}\]

(b) વિઘટન

\[\ce{ZnCO3(s) ->[heat] ZnO(s) + CO2(g)}\]

(c) સંયોગીકરણ

\[\ce{H2(g) + Cl2(g) -> 2HCl(g)}\]

(d) વિસ્થાપન

\[\ce{Mg(s) + 2HCl(aq) -> MgCl2(aq) + H2(g)}\]

✅ પરિણામ

ભાગપ્રકાર
(a)દ્વિવિસ્થાપન
(b)વિઘટન
(c)સંયોગીકરણ
(d)વિસ્થાપન

✅ ઉષ્માક્ષેપક પ્રક્રિયા

જે પ્રક્રિયામાં નીપજ સાથે ઉષ્મા બહાર પડે તેને ઉષ્માક્ષેપક પ્રક્રિયા કહે છે.

\[\ce{CaO + H2O -> Ca(OH)2}\; + \;\text{ઉષ્મા}\]

✅ ઉષ્માશોષક પ્રક્રિયા

જે પ્રક્રિયામાં આસપાસમાંથી ઉષ્મા શોષાય તેને ઉષ્માશોષક પ્રક્રિયા કહે છે.

\[\ce{CaCO3 ->[heat] CaO + CO2}\]

💡 યાદ રાખો

ઉષ્માક્ષેપકમાં પાત્ર ગરમ લાગે; ઉષ્માશોષકમાં પાત્ર ઠંડું લાગી શકે અથવા સતત ગરમી આપવી પડે.

📘 આપેલ વિચાર

પાચનથી મળેલો ગ્લુકોઝ કોષોમાં ઑક્સિજનની હાજરીમાં ધીમે ઑક્સિડાઇઝ થાય છે.

✏️ સમીકરણ

\[\ce{C6H12O6 + 6O2 -> 6CO2 + 6H2O}\; + \;\text{ઊર્જા}\]

✅ અંતિમ જવાબ

શ્વસન દરમિયાન ઊર્જા (ATP અને ઉષ્મા) મુક્ત થાય છે; તેથી શ્વસન ઉષ્માક્ષેપક પ્રક્રિયા છે.

🧠 તફાવત

સંયોગીકરણમાં બે કે વધુ પ્રક્રિયકો જોડાઈ એક નીપજ બનાવે છે. વિઘટનમાં એક પ્રક્રિયક તૂટીને બે કે વધુ નીપજ બનાવે છે; તેથી બન્ને વિરુદ્ધ છે.

✏️ સંયોગીકરણનું ઉદાહરણ

\[\ce{CaO + H2O -> Ca(OH)2}\]

✏️ વિઘટનનું ઉદાહરણ

\[\ce{CaCO3 ->[heat] CaO + CO2}\]

✅ નિષ્કર્ષ

સંયોગીકરણ: અનેક → એક; વિઘટન: એક → અનેક.

✏️ (i) ઉષ્મીય વિઘટન

\[\ce{2Pb(NO3)2 ->[heat] 2PbO + 4NO2 + O2}\]

✏️ (ii) પ્રકાશીય વિઘટન

\[\ce{2AgCl ->[sunlight] 2Ag + Cl2}\]

✏️ (iii) વિદ્યુતીય વિઘટન

\[\ce{2H2O ->[electricity] 2H2 + O2}\]

✅ પરિણામ

ત્રણેયમાં એક સંયોજન ઊર્જા મેળવી બે કે વધુ પદાર્થોમાં તૂટે છે.

📊 તફાવત

વિસ્થાપનદ્વિવિસ્થાપન
વધુ ક્રિયાશીલ તત્ત્વ ઓછા ક્રિયાશીલ તત્ત્વને તેના સંયોજનમાંથી દૂર કરે છે.બે સંયોજનો વચ્ચે આયનો/મૂલકોની આપલે થાય છે.
એક તત્ત્વ + એક સંયોજનબે સંયોજનો

✏️ વિસ્થાપન

\[\ce{Zn + 2AgNO3 -> Zn(NO3)2 + 2Ag}\]

✏️ દ્વિવિસ્થાપન

\[\ce{Na2SO4 + BaCl2 -> BaSO4 v + 2NaCl}\]

📋 પ્રક્રિયકો

કોપર ધાતુ અને સિલ્વર નાઇટ્રેટનું જલીય દ્રાવણ.

✏️ સમતોલિત સમીકરણ

\[\ce{Cu(s) + 2AgNO3(aq) -> Cu(NO3)2(aq) + 2Ag(s)}\]

✅ સમજણ

કોપર સિલ્વર કરતાં વધુ ક્રિયાશીલ હોવાથી Ag⁺ને દ્રાવણમાંથી ધાતુરૂપ સિલ્વર તરીકે વિસ્થાપિત કરે છે.

✅ વ્યાખ્યા

દ્રાવણોમાં થતી એવી પ્રક્રિયા જેમાં પાણીમાં અદ્રાવ્ય ઘન પદાર્થ બને તેને અવક્ષેપન પ્રક્રિયા કહે છે; બનતો ઘન અવક્ષેપ કહેવાય છે.

✏️ ઉદાહરણ

\[\ce{AgNO3(aq) + NaCl(aq) -> AgCl(s) v + NaNO3(aq)}\]

👀 અવલોકન

AgClનો સફેદ અવક્ષેપ મળે છે. આ દ્વિવિસ્થાપન પણ છે.

✅ (a) ઑક્સિડેશન

કોઈ પદાર્થમાં ઑક્સિજન ઉમેરાય અથવા હાઇડ્રોજન દૂર થાય તેને ઑક્સિડેશન કહે છે.

\[\ce{2Mg + O2 -> 2MgO}\]
\[\ce{2Cu + O2 -> 2CuO}\]

✅ (b) રિડક્શન

કોઈ પદાર્થમાંથી ઑક્સિજન દૂર થાય અથવા હાઇડ્રોજન ઉમેરાય તેને રિડક્શન કહે છે.

\[\ce{CuO + H2 -> Cu + H2O}\]
\[\ce{ZnO + C -> Zn + CO}\]

💡 જોડાયેલી પ્રક્રિયા

ઘણી પ્રક્રિયામાં ઑક્સિડેશન અને રિડક્શન સાથે થાય છે; તેને રેડૉક્સ પ્રક્રિયા કહે છે.

🎯 ઓળખ

ચળકતું કથ્થાઈ તત્ત્વ X = કોપર (Cu).
કાળો સંયોજન = કોપર(II) ઑક્સાઇડ (CuO).

✏️ સમીકરણ

\[\ce{2Cu(s) + O2(g) ->[heat] 2CuO(s)}\]

✅ નિષ્કર્ષ

કોપરનું ઑક્સિડેશન થાય છે.

✅ જવાબ

રંગનું પડ લોખંડને હવા અને ભેજના સીધા સંપર્કથી અલગ રાખે છે. ઑક્સિજન અને પાણી ન મળતાં લોખંડનું ક્ષારણ/કાટ લાગવાની પ્રક્રિયા ધીમી પડે અથવા અટકે છે.

💡 પરીક્ષા મુદ્દો

કાટ માટે લોખંડ, ઑક્સિજન અને ભેજ — ત્રણેય જરૂરી છે. રંગ તેમામાંથી ઑક્સિજન અને ભેજનો સંપર્ક રોકે છે.

✅ જવાબ

તેલ અને ચરબીનું ઑક્સિજનથી ઑક્સિડેશન થતાં ખોરાપણું આવે છે. નાઇટ્રોજન નિષ્ક્રિય વાયુ છે; તે થેલીમાંથી ઑક્સિજનને દૂર રાખે છે અને ઑક્સિડેશન ધીમું કરે છે.

🧠 પરિણામ

સ્વાદ અને ગંધ લાંબા સમય સુધી જળવાય છે અને ખોરાકનું શેલ્ફ-લાઇફ વધે છે.

✅ (a) ક્ષારણ

હવા, ભેજ અથવા અન્ય પર્યાવરણીય પદાર્થોની અસરથી ધાતુ ધીમે-ધીમે પોતાના સંયોજનમાં ફેરવાઈ નષ્ટ થાય તેને ક્ષારણ કહે છે.

ઉદાહરણ: લોખંડ પર લાલ-કથ્થાઈ કાટ લાગવો.

✅ (b) ખોરાપણું

તેલ/ચરબીયુક્ત ખોરાકનું ઑક્સિડેશન થઈ તેની ગંધ અને સ્વાદ બગડે તેને ખોરાપણું કહે છે.

ઉદાહરણ: ખુલ્લી રાખેલી તળેલી ચિપ્સમાં ખરાબ ગંધ આવવી.

💡 નિવારણ

ક્ષારણ: રંગ, ગેલ્વેનાઇઝેશન, તેલ/ગ્રીસ.
ખોરાપણું: હવાબંધ પેકિંગ, રેફ્રિજરેશન, ઍન્ટિ-ઑક્સિડન્ટ અને નાઇટ્રોજન પેકિંગ.
